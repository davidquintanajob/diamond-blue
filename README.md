# ⚙ Technologies

| Technology | Purpose |
|------------|---------|
| **Nuxt 4** | Full-stack Vue framework used to build the application |
| **Vue 3** | Progressive JavaScript framework |
| **Tailwind CSS 4** | Utility-first CSS framework for responsive UI |
| **Vite** | Lightning-fast development server and build tool |
| **Pinia** | State management for Vue applications |
| **Nuxt Image** | Automatic image optimization and responsive loading |
| **Nuxt Sitemap** | Automatic sitemap generation for SEO |
| **Nuxt Robots** | Search engine crawling configuration |
| **Nuxt OG Image** | Open Graph image generation for social sharing |
| **Nuxt Fonts** | Optimized web font loading |
| **Nuxt Icon** | Icon management with modern icon collections |
| **VueUse** | Collection of reusable Vue Composition utilities |
| **Google Maps JavaScript API** | Interactive maps and location services |
